Research Notes

Problem

Homestay owners receive reviews from multiple platforms and face difficulty analyzing them manually.

Existing Solutions

- Google Review Analysis
- Booking.com Reviews
- Tripadvisor Feedback

Technologies Studied

- Sentiment Analysis
- Natural Language Processing
- Gemini API
- React.js

Themes Identified

- Food
- Cleanliness
- Host Behavior
- Location
- Experience
- Value for Money

Proposed Solution

An AI-powered web application that classifies reviews and generates management responses.
