# AI-Powered Guest Review Sentiment Classifier

## Overview
This project is being developed as part of the TBI-GEU Summer Internship Program 2026.

The system analyzes guest reviews from platforms such as Google Reviews, Booking.com, and TripAdvisor using Artificial Intelligence.

## Features
- Sentiment Analysis
- Theme Detection
- AI Generated Responses
- Analytics Dashboard

## Technology Stack
- React.js
- Node.js
- Express.js
- Firebase
- Gemini API
